Hexo+Github

Posted on 2017-02-22

具体步骤参考： http://www.ciaoshen.com/2016/02/28/githubHexo-1/
http://baixin.io/2015/08/HEXO搭建个人博客/
官方网站： https://hexo.io/zh-cn/docs/commands.html

1. 安装Node.js, git

mac安装了Xcode，git就已经安装好了。

2. 安装Hexo

sudo npm install -g hexo-cli
hexo init Hexo
cd Hexo
npm install

3. 设置github

github新建repo，hostname.github.io

4. 修改_config.yml

deploy:
type: git
repo: https://github.com/cbl09/cbl09.github.io.git
branch: master
message: Hexo update!

5.生成与同步

hexo g #将.md文件转换成.html静态页面
hexo s #页面预览
hexo d #github同步

可以直接用命令 hexo d -g

6. 页面主题

#主题选择

#主题设置
我选择的是Next主题，设置参考 http://www.arao.me/2015/hexo-next-theme-optimize-base/

刚搭了个很粗糙的版本，接下来看看域名，Markdown语法，主题等，再进行完善。

常见问题解决

1.hexo d -g 之后页面没有变化 –> 先 hexo clean 然后再执行 hexo d -g
2.categories 和 tags 页面404 –> 

$ hexo new page tags
$ hexo new page categories
$ vim source/tags/index.md
ADD LINE $ type: "tags"
$ vim source/categories/index.md
ADD LINE $ type: "categories"
